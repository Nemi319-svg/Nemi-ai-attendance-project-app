# 📸 Snap Class App

A modern, interactive classroom management system built with Python and Streamlit. Snap Class provides dedicated dashboards for teachers and students, integrating advanced features like face recognition and secure authentication to streamline educational tasks.

## ✨ Features

* **Role-Based Access:** Dedicated login and dashboard screens for Teachers and Students.
* **Face Recognition System:** Integrated facial recognition for attendance and verification using `face_recognition` and `dlib`.
* **Secure Authentication:** Passwords and user credentials secured using `bcrypt`.
* **Database Management:** Seamless and real-time backend database integration using `Supabase`.
* **Interactive UI:** Clean and responsive user interface built completely with `Streamlit`.
* **QR Code Integration:** Dynamic QR code generation capabilities using `segno`.

## 🛠️ Tech Stack

* **Frontend & Framework:** Streamlit
* **Backend & Database:** Supabase
* **Machine Learning & Vision:** Scikit-learn, Face_recognition, OpenCV, Dlib
* **Data Processing:** NumPy, Pandas
* **Security:** Bcrypt
